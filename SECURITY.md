# Security Policy

## Supported Versions

| Version   | Supported          |
| --------- | ------------------ |
| > 0.0.9   | :white_check_mark: |
| < 0.0.9   | :x:                |

## Reporting a Vulnerability

To report any vulnerability just open an issue with it here on github. 

If you want to submit any fixes just open a pr.
